# USDT Staking Vault ‚Äì EDUChain

---

## üéØ Goals

1. **Increase Stablecoin TVL on EDUChain**
   - Attract USDT/USDC from Arbitrum and other chains.
   - Users mint OCUSD on EDUChain using bridged stables.
   - Consider restricting OCUSD withdrawals to EDUChain to retain liquidity.

2. **Deliver Sustainable Yield for Stablecoin Holders**
   - Deploy bridged stables into off-chain or L2 strategies (e.g., Ethena, USDX).
   - Return yield via `REWARDER_ROLE` calls to on-chain vaults.

---

## üìù Product Concept

Users deposit USDT/USDC ‚Üí mint **OCUSD** ‚Üí stake OCUSD ‚Üí earn yield from externally deployed strategies.

Model inspired by [Ethena](https://www.notion.so/Ethena-Fork-EDUChain-1fa4928fd54f80e88bf7d39fd40af362?pvs=21) and USDX.

---

## üîë Key Features

- **OCUSD (ERC20 Stablecoin)**
  - Minted from USDT or USDC
  - Redeemable with cooldown

- **sOCUSD (ERC4626 Vault Token)**
  - Capital-appreciating yield-bearing token
  - Cooldown & claim flow

- **Manual Yield Streaming**
  - Admin/multisig streams yield via `transferInRewards()`
  - Controlled via `REWARDER_ROLE`

- **Governance Configs**
  - Fees, minters, pause, blacklist, etc.

## üîß Smart Contract Requirements

### OCUSD (ERC20)

**P0 (Must Have):**
- Mintable by permissioned address
- Role-based minter system
- Burn/redeem function
- Blacklist, pause, governance parameters

**P1 (Nice to Have):**
- OFT-compatible (LayerZero)

---

### Minting / Sales Contract

**P0:**
- Methods: `buy()`, `claim()`, `redeem()`, `cooldown()`
- Fee config
- Cooldown settings
- Whitelist of accepted stables (USDT/USDC)

Example: USDX Sale Contract  
`0xb45c42Fbf8AF8Df5A1fa080A351E9B2F8e0a56D1`

---

### ERC4626 Vault (sOCUSD / sEDU)

**P0 (Must Have):**
- Cooldown start: `cooldownAssets()` / `cooldownShares()`
- Withdraw: `claim()` or `unstake()`
- Previews: `previewWithdraw()`, `getAmountByShares()`
- Yield injection via `REWARDER_ROLE`
- Configurable governance params

## üîÑ User Flow

### For USDT/USDC Users

1. Deposit stable ‚Üí Mint OCUSD  
2. Stake OCUSD ‚Üí Receive sOCUSD  
3. Earn external yield  
4. Cooldown ‚Üí Claim ‚Üí Withdraw

### For EDU Users

2. Use sEDU across DeFi
